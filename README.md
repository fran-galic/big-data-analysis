# Big Data Analysis Labs (FER)

Laboratory exercises for the course "Analiza velikih skupova podataka" at the Faculty of Electrical Engineering and Computing (FER), University of Zagreb (2025/2026).

## Course Overview

This repository contains implementations of core algorithms and systems used in large-scale data analysis, including:

* Similarity search (LSH, MinHash, SimHash)
* Graph mining (PageRank, HITS, community detection)
* Frequent itemset mining (Apriori, PCY, SON)
* Data stream processing
* Clustering algorithms for large datasets (BFR, CURE)
* Recommender systems (collaborative filtering)
* Dimensionality reduction and feature selection
* MapReduce programming model

## Learning Goals

Through these labs, the goal is to:

* Understand what constitutes a big data problem
* Apply the MapReduce paradigm
* Design scalable systems for:

  * similarity search
  * frequent pattern mining
  * graph ranking
  * recommendation systems
* Work with streaming data
* Implement algorithms that scale with data size
